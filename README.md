# Student register

Questo programma di esercitazione gestisce i dati di un elenco di studenti:
- Nome
- Cognome
- Classe
- Sezione

## Tecnologie utilizzate
Vettore statico con numero fisso di elementi, dove ogni elemento
è una struct che contiene i dati.

## Salvataggio su file
Tutti i dati del registro vengono salvati e letti dal file di testo `registro.txt`. 
Il programma legge e scrive i dati carattere per carattere, quindi è compatibile con qualsiasi editor di testo.

## Funzionalità
Il programma è provvisto di un menù testuale per gestire il registro:

1. Inserisci studente
2. Cerca studente per cognome
3. Visualizza registro
4. Ordina registro per cognome
5. Elimina studente
6. Modifica dati studente
7. Salva registro su file

## Come usare il programma
1. Scarica la repository.
2. Compila il file `main.cpp` usando un compilatore C++.
3. Esegui il programma (`./esercizi` su Linux/Mac o `esercizi.exe` su Windows).
4. Scegli l’operazione da svolgere tramite il menù.
